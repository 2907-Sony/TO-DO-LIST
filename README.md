## To-Do List

This is a simple To-Do List web application that allows users to add and delete items from a list.
It is built using basic web technologies and focuses on simple DOM manipulation.

This project is created for practice and learning purposes.

## Features

--Add new items to the to-do list
--Delete items from the list
--Instant updates without page reload
--Simple and clean interface

## Technologies Used

--HTML
--CSS
--JavaScript

## How to Run

--Download or clone the repository
--Open index.html in any web browser
--Add items to the list and remove them as needed
